Extends the WordPress Classic Editor with colors, styles, buttons, Font Awesome icons, and an emoji picker.

== Description ==
Dieses Plugin erweitert den WordPress Classic Editor um:
* Font-Auswahl
* Farben
* Button-Styles
* Font Awesome Icons
* Emoji Picker

== Installation ==
1. Plugin hochladen
2. Aktivieren
3. Einstellungen unter „Einstellungen → Editor Font Tools“
